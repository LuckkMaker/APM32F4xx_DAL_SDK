# APM32F4xx DAL SDK

## Overview

This repository is based on the **APM32F4xx DAL SDK** provided by Geehy Semiconductor, adapted for use with the [Betaflight](https://github.com/betaflight/betaflight) flight controller firmware project.

The APM32F4xx DAL (Driver Abstraction Layer) SDK provides a comprehensive set of peripheral drivers and middleware for the APM32F4xx series of ARM Cortex-M4 microcontrollers.

Key components included:
- **DAL Drivers** — Hardware abstraction layer drivers for all APM32F4xx peripherals (GPIO, UART, SPI, I2C, TIM, ADC, DMA, USB, etc.)
- **DDL Drivers** — Direct low-level register access drivers for performance-critical operations
- **CMSIS** — ARM Cortex-M4 core support and device definitions
- **USB Middleware** — USB Device and Host stack libraries
